# OsteoJP Loops
One loop, one file. Filename: <id>-<slug>.md (example 0023-therapist-service-mapping.md).
Each loop file carries the 7 Loop Package fields: scope and ground truth, ordered steps, machine-verifiable definition of done, verification with pasted evidence, restrictions and scope boundary, halt-loud protocol, report-back format.
A loop is dispatchable only when its row in BACKLOG.md is READY. Status is owned by BACKLOG.md, not by the loop file.
