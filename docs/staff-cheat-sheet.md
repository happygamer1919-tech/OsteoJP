## O que mudou (vs Fisiozero)

| Fisiozero | OsteoJP |
|---|---|
| Lista de utentes | **Pacientes** (barra lateral) |
| Ficha do utente | **Perfil do utente** |
| Evento / marcação | **Marcação** (na Agenda) |
| Tipo de evento (RPG, osteopatia…) | **Serviço** (escolhido na marcação) |
| Nota clínica | **Ficha clínica** (dentro de um Episódio) |
| Histórico de consultas | **Episódios** (no perfil do utente) |
| Configurações | **Administração** |

## Tarefas do dia-a-dia

**Marcar uma consulta**
Agenda → **+ Nova Marcação** → preenche utente, serviço, terapeuta, data, hora → Guardar

**Pesquisar um utente**
Pacientes → barra de pesquisa → nome, NIF ou telefone

**Criar um novo utente**
Pacientes → **Novo Utente** → campos obrigatórios: nome, data de nascimento, sexo → Criar Utente

**Iniciar uma consulta (novo episódio)**
Perfil do utente → **+ Novo Episódio** → dentro do episódio → **+ Nova ficha neste episódio** → seleciona modelo → preenche → Guardar

**Remarcar uma consulta**
Agenda → clica na marcação → **Remarcar** → nova data/hora → confirmar

**Cancelar uma consulta**
Agenda → clica na marcação → **Cancelar** (fica no histórico, nunca apagada)

---

## Regras importantes

- **Não apagues utentes duplicados** — usa "Fundir duplicado" ou pede ajuda
- **Ficha assinada = bloqueada para sempre** — verifica antes de assinar
- **Dúvida sobre dados antigos?** — consulta o Fisiozero (acesso de leitura mantido durante a transição)
- **Erro inesperado?** — tira captura de ecrã e reporta à administração
