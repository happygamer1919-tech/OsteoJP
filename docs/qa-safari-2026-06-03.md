## Bug found

### BUG-14 — VAT rate set to 23% in admin settings
**Severity:** 🔴 Critical (fiscal risk)
**Route:** `/admin/settings` → Faturação → Taxa de IVA (%)
**Description:** The VAT rate field shows **23** (Portugal's standard VAT rate). OsteoJP is exempt from VAT under art. 9.º n.º 1 CIVA (health services exemption). If invoices are issued with 23% VAT applied, this is a fiscal error with legal implications.
**Expected:** 0 (exempt)
**Action required:** Either lock the field at 0 (recommended — clinic has no non-exempt services), or João Pedro to manually set it to 0 immediately. Do not issue any invoices until this is resolved.
